## Concept

GameBox est une plateforme sociale dédiée aux jeux vidéo, inspirée du concept de Letterboxd (pour le cinéma) mais adaptée à l'univers gaming.

L'objectif : créer un espace où les joueurs peuvent découvrir, noter, critiquer et partager leurs expériences de jeu.

## Fonctionnalités

### Pages de jeux
Chaque jeu dispose de sa page dédiée avec :
- Informations détaillées (développeur, genre, date de sortie)
- Note moyenne de la communauté
- Threads interactifs générés automatiquement

### Système de critiques
- Notes sur 5 étoiles
- Critiques textuelles
- Likes et commentaires sur les critiques

### Forums et Messagerie
- Forums de discussion par jeu
- Messagerie privée entre utilisateurs
- Notifications en temps réel

### Listes personnalisées
- Création de listes thématiques
- Partage public ou privé
- Import/export de données

## Stack Technique

- **Frontend** : Angular
- **Backend** : Java (Spring Boot)
- **Base de données** : PostgreSQL
- **API** : REST

## Défis techniques

- Génération de threads : algorithme pour créer des discussions automatiques basées sur l'activité
- Performance : optimisation des requêtes pour gérer un grand nombre de jeux et d'utilisateurs
- UX : interface intuitive pour naviguer dans un catalogue de milliers de jeux

## Statut

Projet en cours de développement, avec les fonctionnalités principales opérationnelles.
